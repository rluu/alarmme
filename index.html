    Alarm clock application

    Project maintained by rluu Hosted on GitHub Pages

    Description:

    AlarmME ('Alarm Made Easy') is an alarm clock program which plays any media file of choice as the alarm. It features re-alarming intervals and auto-saving of the user settings.


    Features:

    • Customizable re-alarming time intervals.
    • Auto-saving of user settings.
    • Built-in support for several media players:
      • Windows Media Player
      • Winamp
      • XMMS
      • Xine
      • Mplayer
      • or any custom application

    Technologies:

    This is a cross-platform application that can run on the Windows, Mac and Linux platforms.
    Some of the technologies utilized:

    • C++
    • Qt 3.2

    Downloads:

    Installable packages are available for the Windows platforms only:

    • Windows:
      • AlarmME-setup-1.0.0.exe
      • QtLibrary-setup-3.2.1.exe (required dependency)
